# MCP Server Configurations

This directory contains configuration templates for Microsoft Enterprise MCP Servers.

## Files

- `copilot-enterprise-settings.json` - Copilot Enterprise MCP configuration template
- `teams-bot-mcp-config.json` - Teams Bot integration configuration

## Usage

1. Copy the appropriate config file
2. Replace placeholder values (YOUR_SUBSCRIPTION_ID, etc.)
3. Apply to your environment

See main README.md for setup instructions.
